# Costa Rica Itinerary — May 14–25, 2026

Interactive single-page travel itinerary for a 12-day Costa Rica trip. Built with React + Vite + Tailwind CSS + Framer Motion.

## Setup

```bash
npm install
npm run dev
```

Opens at `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```

## Stack

- **React 18** + **Vite 5** — fast dev, HMR
- **Tailwind CSS 3** — utility-first styling
- **Framer Motion** — spring animations, scroll progress
- **Lucide React** — SVG icons
- **Google Fonts** — Playfair Display, Outfit, JetBrains Mono

## Features

- Live countdown timer to departure (May 14, 2026)
- 12 expandable day cards with drive banners, callouts, and options grids
- Horizontal scrollable timeline strip with color-coded day pills
- SVG budget donut chart with animated draw
- Boarding-pass style flight cards
- Booking checklist with localStorage persistence
- Sticky frosted-glass nav with scroll-spy
- Scroll progress bar
- Full mobile responsive design
